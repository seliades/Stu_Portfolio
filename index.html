Discover Live Music

Visit Site / GitHub Repo

Plan your next concert with this music discovery web application which combines the powers of Spotify and SeatGeek APIs. Search your local area to find out what bands are going to be in town, listen to those artists' top 3 tracks on Spotify to discover your next favorite band, and find the perfect tickets on SeatGeek.

Muppets in GIFs

Visit Site / GitHub Repo

This page is a tribute to the Muppets. Pressing any of the buttons will return 10 GIFs of that Muppet, and you can also use the Add Muppet button to add more Muppets! The GIFs will begin in still mode, and you can toggle back and forth between still and active by clicking each GIF.

Animal Collectives Trivia

Visit Site / GitHub Repo

This timed trivia game tests your knowledge of the collective nouns for groups of different animals. You've heard of a pack of wolves or a school of fish, but what do you call a group of porcupines? Answer as many correctly as you can, you only get 8 seconds per questions!

Imagination Station

Visit Site / GitHub Repo

This web app tracks the schedule of any train you can imagine. First give your train a name and destination. Then just tell us when the first train runs, how often it runs and you will see what time the next train will arrive, and how long you have left to wait.

LIRI Node App

GitHub Repo

This LIRI Node application is a Language Interpretation and Recognition Interface that runs through the command line and utlizes JavaScript and several Node modules and APIs. LIRI accepts several commands and returns relavent information to the user based on the command and search query used.
